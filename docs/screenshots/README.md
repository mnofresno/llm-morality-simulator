# Screenshots Documentation

This directory contains screenshots of the LLM Behavior Lab application interface, showcasing its main features and functionality.

- Shows the landing page of the LLM Behavior Lab
- Displays the application title, description, and navigation tabs
- Demonstrates the clean, modern UI design

### 02-sidebar-configuration.png
**Sidebar Configuration Panel**
- Shows the configuration sidebar with model selection options

### 03-experiments-list.png
**Experiments List Tab**
- Shows the list of all executed experiments

**View Results Tab**

## Usage

These screenshots are used in the main README.md to help users understand the application's interface and features. They provide visual context for the documentation and help new users quickly understand how to use the application.

## Capturing New Screenshots

To capture new screenshots:

1. Start the Streamlit application:
   ```bash
   streamlit run streamlit_app.py
   ```

2. Navigate to the feature you want to capture

3. Use a screenshot tool or browser extension to capture the full page

4. Save the screenshot with a descriptive name following the numbering convention

5. Update this README with a description of the new screenshot
